# LLM ChatBot App – SIT305 Task 8.1

A simple Android chat application that lets users log in with a username and have a conversation with an AI chatbot.

## Features
- Username login
- Real-time chat with an AI assistant
- Chat history saved locally on the device
- Message timestamps

## How to Run
1. Clone the repository
2. Open the project in Android Studio
3. Start the backend server with `node server.js`
4. Run the app on an emulator or device

## Tech Stack
- Java (Android)
- Node.js (backend)
- Room (local database)
- Retrofit (networking)
